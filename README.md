# Node.js E-Commerce REST API

A fully functional E-Commerce API built with Express, Sequelize, and PostgreSQL.

## Features
- User Auth (JWT)
- Products & Categories CRUD
- Image Upload (Cloudinary)
- Shopping Cart & Orders
- Swagger API Docs
- Unit Testing (Postman)

## Setup

```bash
npm install
cp .env.example .env
npm run dev
